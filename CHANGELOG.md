# Change log

All notable changes to this project will be documented in this file.
This project adheres to [Semantic Versioning](http://semver.org/).

## [1.1.0] - 2026-03-31

* Add LinkedIn provider support.
* Connect LinkedIn accounts via OAuth 2.0 (authorization code flow) using user-configured OAuth apps.
* Support scheduling and publishing LinkedIn posts with media uploads.
* Support LinkedIn-specific provider overrides and per-account text overrides for multi-account scheduled posts.
* Update API docs and README for LinkedIn account setup, posting, and media limits.

## [1.0.0] - 2026-02-26

* Initial Release of BlueWing Social Scheduler.
* Posting to Bluesky
* Posting to X
* Scheduling with media
* Manage and share access to accounts
* Built in API for programmatic scheduling.
